# Estruturas de Dados com TDD
Estruturas de Dados com Testes Unitários usando UnitTestCPP

# Como usar
Clone este repositório, dê checkout no branch 'dojo' e implemente as funções para que todos os casos de testes passe.
Os casos de testes estão comentados. Descomente o caso de teste de uma função, implemente-a e, após o caso de teste para essa função passar, descomente outro caso de teste, implemente a função e, após o caso de teste passar descomente outro e assim por diante.
